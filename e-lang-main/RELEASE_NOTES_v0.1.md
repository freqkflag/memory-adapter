# E-Lang v0.1.0 — Initial Public Release

This is the first public release of **E-Lang**, a formal symbolic language for
representing emotional experience prior to linguistic labeling.

## What This Release Includes

- Frozen core grammar and axis set
- Canonical specification (`ELANG_SPEC.md`)
- Formal ABNF grammar (`ELANG.abnf`)
- Macro library and journal template
- Reference research manuscript
- Governance documents (AGENTS, CONTRIBUTING, ROADMAP)
- CI workflow for documentation and tests

## What Is Frozen in v0.1

- Axis set and order: `V,A,C,S,K,D,B,N`
- Canonical state syntax: `E{V,A,C,S,K,D,B,N}`
- Operator set and precedence
- Grammar as defined in `ELANG.abnf`

Breaking changes to the above require a **major version bump**.

## Explicitly Out of Scope

- Diagnosis or therapy
- Emotion labeling or taxonomies
- Automated affect inference
- Normative emotional scoring

## What's Next (v0.2)

- Reference parser and validation tooling
- Improved developer documentation
- Visual and tutorial materials
- Research replication support

E-Lang is infrastructure.  
Stability and clarity take precedence over expansion.
